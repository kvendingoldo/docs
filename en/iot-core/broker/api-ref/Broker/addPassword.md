# IoT Core Broker Service, REST: Broker.AddPassword {#AddPassword}

Adds password for the specified broker.

## HTTP request

```
POST https://{{ api-host-iot-broker }}/iot-broker/v1/brokers/{brokerId}/passwords
```

## Path parameters

#|
||Field | Description ||
|| brokerId | **string**

Required field. ID of the broker to add a password for.

To get a broker ID make a [BrokerService.List](/docs/iot-core/broker/api-ref/Broker/list#List) request. ||
|#

## Body parameters {#yandex.cloud.iot.broker.v1.AddBrokerPasswordRequest}

```json
{
  "password": "string"
}
```

#|
||Field | Description ||
|| password | **string**

Passwords for the broker.

The password must contain at least three character categories among the following: upper case latin, lower case latin, numbers and special symbols. ||
|#
