# Yandex Cloud Marketplace License Manager for SaaS, REST: Lock.Get {#Get}

Returns the specified subscription lock.

## HTTP request

```
GET https://marketplace.{{ api-host }}/marketplace/license-manager/saas/v1/locks/{lockId}
```

## Path parameters

#|
||Field | Description ||
|| lockId | **string**

Required field. ID of the subscription lock. ||
|#
