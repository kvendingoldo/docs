# Video API, REST: StreamLine.Get {#Get}

Returns the specific stream line.

## HTTP request

```
GET https://video.{{ api-host }}/video/v1/streamLines/{streamLineId}
```

## Path parameters

#|
||Field | Description ||
|| streamLineId | **string**

Required field. ID of the line. ||
|#
