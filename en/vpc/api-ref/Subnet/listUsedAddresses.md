# Virtual Private Cloud API, REST: Subnet.ListUsedAddresses {#ListUsedAddresses}

List used addresses in specified subnet.

## HTTP request

```
GET https://vpc.{{ api-host }}/vpc/v1/subnets/{subnetId}/addresses
```
